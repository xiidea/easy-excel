# Formula function coverage

`getCalculatedValue()` / `toArray(calculateFormulas: true)` delegate to
[excelize](https://github.com/qax-os/excelize)'s formula engine. This table
probes every function PhpSpreadsheet's calculation engine registers
(extracted from `Calculation/FunctionArray.php`) against that engine —
regenerate with `go run ./tools/formula-coverage > ../FORMULAS.md` from
`extension/`.

**466 of 529** PhpSpreadsheet functions are available (63 missing).

Semantics of individual functions may still differ in edge cases
(see COMPAT.md); cached results in loaded files are always preferred.
